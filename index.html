Ready-to-Use AI Prompts

Copy-paste prompts for ChatGPT, Claude, and other AI assistants. Tested and optimized.

📝 Writing

Blog Post Generator

Write a comprehensive blog post about [TOPIC]. Structure it with an engaging introduction, 5 main sections with subheadings, practical examples, and a conclusion with actionable takeaways. Target audience: [AUDIENCE]. Tone: [professional/casual/educational].
💼 Business

Email Reply Assistant

You are a professional email assistant. I received this email: "[PASTE EMAIL]". Write a polite, professional reply that: 1) Acknowledges their message, 2) Addresses their main points, 3) Proposes next steps. Keep it concise (under 150 words).
🎓 Learning

Concept Explainer

Explain [CONCEPT] like I'm [age/level]. Use: 1) A simple analogy from everyday life, 2) A clear definition in 2 sentences, 3) Three real-world examples, 4) Common misconceptions, 5) A quick quiz question to test understanding.
💻 Coding

Code Review & Debug

Review this code for: 1) Bugs and logic errors, 2) Security vulnerabilities, 3) Performance issues, 4) Best practice violations. For each issue found, explain WHY it's a problem and provide the corrected code. Code: [PASTE CODE]
🎨 Creative

AI Image Prompt Builder

Create a detailed image generation prompt for: [DESCRIBE WHAT YOU WANT]. Include: art style (photorealistic/illustration/oil painting), lighting (golden hour/studio/dramatic), camera angle (close-up/wide/aerial), mood, color palette, and specific details. Output as one continuous paragraph.
📊 Analysis

Data Summarizer

Analyze this data/text and provide: 1) Executive summary (3 sentences), 2) Key findings (bullet points), 3) Trends or patterns noticed, 4) Actionable recommendations, 5) Questions that need further investigation. Data: [PASTE DATA]
